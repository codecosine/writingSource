## 前言

导入模块和库,一直是各类语言中不能分割的一部分。
在JavaScript发展中,衍生出模块的思想。
标志性就是requirejs,seajs之类的工具包产生，在ES6之前，require统治了所有模块化变成。也因此产生了很多AMD,CMD的规范

## 我简单总结先

import是ES6的规范
import在未被支持前，都是使用babel进行编译
因此 require是运行时， import 是编译时
    require时赋值过程, import 是解构过程
